# ZUKO-MD Social Media Downloader

A Flask-based web application for downloading videos from various social media platforms without watermarks.

## Features
- Download videos from 13+ social media platforms
- No watermarks on downloaded videos
- Batch download support (up to 10 URLs)
- Multiple format selection
- Clean, responsive UI
- Easy deployment on Render

## Supported Platforms
- YouTube
- TikTok
- Instagram
- Facebook
- Twitter/X
- Pinterest
- Reddit
- Likee
- Snapchat
- LinkedIn
- Threads
- Dailymotion
- Vimeo
